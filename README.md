The Student Performance Analyzer is a web-based application developed using Python Flask that allows teachers to upload student marks in CSV format and automatically analyze academic performance.
Objectives

To analyze student performance efficiently

To automate calculation of averages, highest and lowest marks

To identify subject-wise toppers

To provide a user-friendly web interface for teachers

⚙️ Technologies Used

Python

Flask (Web Framework)

Pandas (Data Analysis)

HTML & Bootstrap (Frontend)

Git & GitHub (Version Control)

✨ Features

Upload student marks using CSV file

Display student data in rows and columns (table format)

Calculate subject-wise:

Average marks

Highest marks and topper

Lowest marks

Simple and clean web interface

Easily extendable for charts, PDF reports, and login system
